## Installation
`npm install vue-tree-view`  
and in your component  
`import VueTreeView from 'vue-tree-view'`  

## Usage  
`<vue-tree-view :treeData=yourData></vue-tree-view>`

## Props  
| Name | Type | Default | Description |
| --- | --- | :---: | --- |
| treeData | Array | [] | data for rendering tree |
